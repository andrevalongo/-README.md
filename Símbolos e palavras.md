Símbolos
1. Operadores Aritméticos;
+ : Adição
- : Subtração
* : Multiplicação
/ : Divisão real (retorna um valor flutuante/decimal)
// : Divisão inteira (descarta a parte decimal)
% : Módulo (retorna o resto da divisão)
 
2. Operadores Relacionais;
== : Igual a
!= : Diferente de
> : Maior que
< : Menor que
>= : Maior ou igual a
<= : Menor ou igual a
3. Operadores booleanos;
& : AND bit a bit (E)
| : OR bit a bit (OU)
^ : XOR bit a bit (OU Exclusivo)
~ : NOT bit a bit (Negação)

4. Símbolos de Atribuição;
= : Atribuição simples
+= : Atribui e soma
-= : Atribui e subtrai
*= : Atribui e multiplica
/= : Atribui e divide
//= : Atribui e faz divisão inteira
%= : Atribui e calcula o módulo
= : Atribui e calcula a exponenciação
5. Simbolos identificadores de tipos de dados;
" " , ' ' : Strings
[ ] : Listas
( ) : Espressões
{ } : Dicionários 
6. Símbolo identificador de início de bloco
: Dois pontos
